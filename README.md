Travelgram

Um perfil de viagens moderno e visual, inspirado em redes sociais 📸✈️

📌 Sobre o projeto

O Travelgram é uma página web que simula o perfil de uma viajante, exibindo suas informações pessoais e uma galeria com registros de suas viagens ao redor do mundo.

O projeto foi desenvolvido com foco em estrutura HTML semântica e estilização com CSS, ideal para praticar front-end.

🖼️ Layout

O projeto possui:

🔝 Navbar com menu e perfil

👤 Seção de perfil com descrição

📊 Informações de viagem (localização, países, fotos)

🖼️ Galeria de imagens

🔻 Rodapé com informações

🚀 Tecnologias

HTML5

CSS3

Google Fonts (Poppins)
